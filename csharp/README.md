# csharp crypto corpus (synthetic)

Generated test files packed with cryptographic algorithm names
(AES, RSA, SHA-256, ChaCha20, ML-KEM, Dilithium, MD5, DES, ...) for
static-analysis detection coverage. **Not** a real crypto library.

- Some files compile, some intentionally do not.
- Includes fake APIs, dead code, deprecated usage, and misuse.
